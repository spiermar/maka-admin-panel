# Output Patterns

## Template Pattern

Provide templates for output format:

**For strict requirements:**
```markdown
## Report structure

ALWAYS use this exact template:

# [Title]
## Summary
[key findings]

## Recommendations
[items]
```

## Examples Pattern

For quality-dependent output:

```markdown
## Commit message format

Example:
Input: Added user auth
Output: feat(auth): implement JWT-based auth

Follow: type(scope): description
```
